## Circular Queue

 - front와 rear 값이 시계 방향으로 한 칸씩 이동하는 형태
 - 실제 구현에서는 배열 전체 크기를 이용하여 나머지(%) 연산 수행
 -  empty 또는 full 상태인지를 확인하기 위해 원소의 개수를 저장할 변수 필요
